# do1ffe-webseite

Neu gestaltete, mehrseitige Flask-Webseite für DO1FFE mit inhaltlichem Fokus auf Amateurfunk, Softwareprojekte, GitHub-Repositories und Tesla-Dashboard.

## Seitenstruktur

- `/` Startseite mit Überblick
- `/ueber-mich` Hintergrund und Profil
- `/projekte` Projekte, Repositories und Dashboard-Verweise
- `/kontakt` Kontakt und weiterführende Links

## Starten

```bash
pip install -r requirements.txt
python app.py
```

Danach ist die Seite unter `http://0.0.0.0:8020` erreichbar.
